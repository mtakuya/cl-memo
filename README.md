NAME  
    cl-memo is a memo app.

VERSION
    0.0.1
 
INSTALLATION  
    1. install roswell   
    2. `$ ros install mtakuya/cl-memo`  
    3. `$ cd ~/.roswell/local-projects/mtakuya/cl-memo/.roswell`  
    4. `$ ros build memo.ros`  
    5. `$ memo test.txt`
